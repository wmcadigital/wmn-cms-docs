# Major Roadworks and Events

These pages display any current or future roadworks or events which will cause travel disruption. All of these page are added through the CMS.

## How it looks

![major-works-main-hub](../../../public/pages/major-works/main-hub/major-works-main-hub.png)

## Functionality

1. Live Results - All events that are currently within todays date / time
2. Planned Results - All events thats are after todays date / time
3. Filter results by West Midlands area. The area will only appear if there's an event tagged in the area
4. Related Content - Add custom links into this sidebar tile
